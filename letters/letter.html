Placeholders, Randomizers, and Customizers

    Placeholders

  • [[email]]: The recipient's email in all lowercase (e.g., user@domain.com).
  • [[Email]]: The recipient's email with the first letter capitalized (e.g., User@domain.com).
  • [[EMAIL]]: The recipient's email in all uppercase (e.g., USER@DOMAIN.COM).
  • [[user]]: The username (local part) before the `@` in the recipient's email in lowercase (e.g., user from user@domain.com).
  • [[User]]: The username (local part) before the `@` with the first letter capitalized (e.g., User from user@domain.com).
  • [[USER]]: The username (local part) before the `@` in all uppercase (e.g., USER from user@domain.com).
  • [[domain]]: The domain part of the recipient's email in lowercase (e.g., domain.com).
  • [[Domain]]: The domain part of the recipient's email with the first letter capitalized (e.g., Domain.com).
  • [[DOMAIN]]: The domain part of the recipient's email in all uppercase (e.g., DOMAIN.COM).
  • [[smtpdomain]]: The domain part of the sender's email in lowercase (e.g., sender.com).
  • [[Smtpdomain]]: The domain part of the sender's email with the first letter capitalized (e.g., Sender.com).
  • [[SMTPDOMAIN]]: The domain part of the sender's email in all uppercase (e.g., SENDER.COM).
  • [[company]]: Extracted company name based on the recipient's domain logic. Defaults to lowercase.
  • [[Company]]: Extracted company name with the first letter capitalized.
  • [[COMPANY]]: Extracted company name in all uppercase.
  • [[date]]: Inserts the current date in `YYYY-MM-DD` format (e.g., 2024-11-16).
  • [[emailbase64]]: The recipient's email encoded in Base64 format.
  • [[emailbase64=]]: The recipient's email encoded in Base64 format with the trailing `=` removed.
  • [[link]]: A random line from `link.txt`.
  • [[linkbase64]]: A random line from `link.txt` encoded in Base64 format.

    • Randomizers

    Randomizers select random lines from specific files stored in the `randomizers` directory.

  • [[random_city]]: A random city name from `random_city.txt`.
  • [[random_browser]]: A random browser name from `random_browser.txt`.
  • [[random_state]]: A random state name from `random_state.txt`.
  • [[random_hostname]]: A random hostname from `random_hostname.txt`.
  • [[random_firstname]]: A random first name from `random_firstname.txt`.
  • [[random_lastname]]: A random last name from `random_lastname.txt`.
  • [[random_surname]]: A random surname from `random_surname.txt`.
  • [[random_address]]: A random address from `random_address.txt`.
  • [[random_country]]: A random country name from `random_country.txt`.
  • [[random_continent]]: A random continent name from `random_continent.txt`.

    • Customizers

    Customizers select specific lines from files in the `randomizers` directory based on recipient index.

  • [[custom_city]]: A city name from `custom_city.txt`, indexed by recipient order.
  • [[custom_browser]]: A browser name from `custom_browser.txt`, indexed by recipient order.
  • [[custom_state]]: A state name from `custom_state.txt`, indexed by recipient order.
  • [[custom_hostname]]: A hostname from `custom_hostname.txt`, indexed by recipient order.
  • [[custom_firstname]]: A first name from `custom_firstname.txt`, indexed by recipient order.
  • [[custom_lastname]]: A last name from `custom_lastname.txt`, indexed by recipient order.
  • [[custom_surname]]: A surname from `custom_surname.txt`, indexed by recipient order.
  • [[custom_address]]: An address from `custom_address.txt`, indexed by recipient order.
  • [[custom_country]]: A country name from `custom_country.txt`, indexed by recipient order.
  • [[custom_continent]]: A continent name from `custom_continent.txt`, indexed by recipient order.
