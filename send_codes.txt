        <h2>Placeholders</h2>
        <li>
            <strong>[[email]]</strong>: The recipient's email in all lowercase (e.g., user@domain.com).
        </li>
         <li>
            <strong>[[sender]]</strong>: The sender's email in all lowercase (e.g., user@domain.com).
        </li>
        <li>
            <strong>[[Email]]</strong>: The recipient's email with the first letter capitalized (e.g., User@domain.com).
        </li>
        <li>
            <strong>[[EMAIL]]</strong>: The recipient's email in all uppercase (e.g., USER@DOMAIN.COM).
        </li>
        <li>
            <strong>[[user]]</strong>: The username (local part) before the `@` in the recipient's email in lowercase (e.g., user from user@domain.com).
        </li>
        <li>
            <strong>[[User]]</strong>: The username (local part) before the `@` with the first letter capitalized (e.g., User from user@domain.com).
        </li>
        <li>
            <strong>[[USER]]</strong>: The username (local part) before the `@` in all uppercase (e.g., USER from user@domain.com).
        </li>
        <li>
            <strong>[[domain]]</strong>: The domain part of the recipient's email in lowercase (e.g., domain.com).
        </li>
        <li>
            <strong>[[Domain]]</strong>: The domain part of the recipient's email with the first letter capitalized (e.g., Domain.com).
        </li>
        <li>
            <strong>[[DOMAIN]]</strong>: The domain part of the recipient's email in all uppercase (e.g., DOMAIN.COM).
        </li>
        <li>
            <strong>[[smtpdomain]]</strong>: The domain part of the sender's email in lowercase (e.g., sender.com).
        </li>
        <li>
            <strong>[[Smtpdomain]]</strong>: The domain part of the sender's email with the first letter capitalized (e.g., Sender.com).
        </li>
        <li>
            <strong>[[SMTPDOMAIN]]</strong>: The domain part of the sender's email in all uppercase (e.g., SENDER.COM).
        </li>
        <li>
            <strong>[[company]]</strong>: Extracted company name based on the recipient's domain logic. Defaults to lowercase.
        </li>
        <li>
            <strong>[[Company]]</strong>: Extracted company name with the first letter capitalized.
        </li>
        <li>
            <strong>[[COMPANY]]</strong>: Extracted company name in all uppercase.
        </li>
        <li>
            <strong>[[date]]</strong>: Inserts the current date in `YYYY-MM-DD` format (e.g., 2024-11-16).
        </li>
        <li>
            <strong>[[emailbase64]]</strong>: The recipient's email encoded in Base64 format.
        </li>
        <li>
            <strong>[[emailbase64=]]</strong>: The recipient's email encoded in Base64 format with the trailing `=` removed.
        </li>
        <li>
            <strong>[[link]]</strong>: A random line from `link.txt`.
        </li>
        <li>
            <strong>[[linkbase64]]</strong>: A random line from `link.txt` encoded in Base64 format.
        </li>
